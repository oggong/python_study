정규표현식 (Regular Expression)

1. 확장문자
    s :  공백문자 (스페이스, 탭 등 )
    d :  숫자
    D : 숫자가 아닌 문자 -> [^0-9] 와 동일
    w : 알파벳, 숫자, 언더스코어(_) -> [A-Za-z0-9] 비슷
    W : 알파벳, 숫자, 언더스코어(_)가 아닌 문자

2. 특수문자
    * : 0번 이상
    + : 1번 이상
    ? : 0 또는 1개의 문자 매칭
    . : 정확히 1개 문자 매칭

3. 플래스
    g : 매칭되는 모든 값을 찾음
    i : 대소문자 무시
    m : 여러 줄 매칭
